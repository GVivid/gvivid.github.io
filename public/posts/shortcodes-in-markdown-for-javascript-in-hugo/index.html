Shortcodes in Markdown for Javascript in Hugo

July 12, 2025 One-minute read

I wanted to know how to host Javascript applications on this website. After some research, I discovered that you can use shortcodes in markdown files when generating a hugo website. So, you can place your javascript files into the shortcodes folder in your hugo directory, and then use the shortcode syntax right within your markdown file. This might be helpful for creating a portfolio of Javascript applications.
